Freedom of Speech

A principle that supports the freedom of an individual or a community to articulate their opinions and ideas without fear of retaliation, censorship, or sanction. The term "freedom of expression" is sometimes used synonymously but includes any act of seeking, receiving, and imparting information or ideas, regardless of the medium used.

Censorship

The suppression of speech, public communication, or other information, on the basis that such material is considered objectionable, harmful, sensitive, politically incorrect or "inconvenient" as determined by government authorities or by community consensus.

Features

The mission's primary means to ensure censorship-resistant private communications is through an intelligent application router that routes around blocks using unblocked networks and peers as relays while providing a service platform to more easily enable applications to use the following features:

1DN

Invisible Direct Network - an ad-hoc fully decentralized P2P network using TOR, I2P, Bluetooth, WiFi-Direct, Satellites, Full-Spectrum Radio, and LiFi to support private uncensored communications.

Censorship Resistant Routing

Routes around censorship attempts by using unblocked networks and peers as relays.

Decentralized-Services DApp Platform

A decentralized-service platform for developing decentralized applications within a Service-Oriented Architecture (SOA) and a Staged Event-Driven Architecture (SEDA) supporting routing between services using a routing service that implements Enterprise Integration Patterns (EIP) and Decentralized Routing Algorithms. Services can be managed independently of applications.

Decentralized IDentifier

Pluggable, self-sovereign, decentralized, people and/or automation-controlled universal ID, called DID or Decentralized IDentifier using OpenPGP. Only you have the keys. Build reputation with them.

Key Ring

Encryption and Signing keys kept safe on specialized flash drives where once added they can not be read nor changed. Encryption and signing happens on-drive only. OpenPGP used for standards support.

Info Vault

The OS allows for the user to grant access control to their personal information on their device if desired. Access can be revoked when its no longer desired for them to have it. This will end widespread data breaches. The OS runs purely in memory not accessing the hard-drive, only the physical external drives you provide.
